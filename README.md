# 반응 속도 게임

## 1. 개요

### 1-1. 목표

클릭 반응 속도 게임을 즐길 수 있는 웹 애플리케이션을 개발한다.
<br><br>

### 1-2. 기술 스택

- 프론트엔드: React, TypeScript, TailwindCSS
- 백엔드: Node.js, MongoDB
  <br><br>
